Selecting Import Group Actions

Use this view to specify actions to be performed for the leads you import to Saleslogix. You can add a note, add the record to a campaign, create a response or schedule an activity.

To select

  1. When importing leads, after managing duplicate records, click Next.
  2. To add a note, next to Add Note, click Define, and complete the import lead note boxes. Select the Active box.
  3. To add a response not associated to a campaign, next to Add Response, click Define, and complete the import response boxes. Select the Active box.
  4. To add a campaign target response, next to Add Campaign Target click Define and select a campaign. Select the Active box.
  5. To schedule a phone call, click Schedule Phone Call and complete the import lead activity boxes. Select the Active box.
  6. To schedule a meeting, click Schedule Meeting and complete the import lead activity boxes. Select the Active box.
  7. To schedule a to-do, click Schedule To-Do and complete the import lead activity boxes. Select the Active box.
  8. Click Next to go to the next step.
