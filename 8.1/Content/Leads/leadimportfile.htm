Selecting a File to Import

Use this view to select the file that contains the leads you want to import.

To select a file

  1. Open the Import Leads wizard.
  2. Next to the Select a file containing Leads information to start box, click Browse, browse to the list you want to import, and then click Open.
    The file appears in the Current Lead List File box.
  3. Click the Default Owner Find button to select the person or team who will own the imported leads.
  4. Click the Default Lead Source Find button and use the lookup to select how the imported leads were generated.
  5. To add the leads to an ad hoc group, select Add Imported Leads to ad hoc group and do one of the following:
    • To add the imported leads to a new group, select Create a new ad hoc group and enter group name in the box.
    • To add leads to an existing group, select Add Leads to this existing ad hoc group, then select a group from the list.
  6. Click Next to go to the next step.
