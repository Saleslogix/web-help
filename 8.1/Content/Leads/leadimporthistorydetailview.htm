Using the Lead Import History Detail View

Use the Import History Detail view to see detailed information about a lead import.

How Do I?

Open the Lead Import History view
  • Do one of the following:
    • After importing leads, on the Process Request view, click View Detail.
    • From the Lead Import History List view, click the Import Number of the import you want to view.
Open the Lead Import History List view from the Detail view.
  • From the Lead Import History Detail view, click List
    View    .
Stop an import while it is still processing
  • Click Abort.

Information Boxes

The information boxes contain the primary information about the lead import.

  • Import Number - Unique number that identifies the lead import.
  • Description - Name of the list that was imported.
  • Total records - Number of leads that were in the import list.
  • Total records processed - Number of leads that have been processed. If an import is still processing, this shows how many of the records have been processed.
  • Total records imported - Number of leads that were imported.
  • Total errors - Number of leads that were not imported due to errors.
  • Total warnings - Number of warnings that occurred during the import.
  • Total duplicates - Number of duplicates in the import.
  • Total auto merged - Number of records that were merged during the import.
  • Started By - User who started the import.
  • Status - Status of the import. For example, Completed.
  • Start Date - Date the import was started.
  • Complete Date - Date the import was completed.
  • Import Template - Template used during the import.

Detail View Tabs

Each tab shows specific information you can use for account related activities. Click a link to see more information about each tab.
