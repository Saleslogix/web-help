Adding an Import Group Action Response

Use this view to add a response to each imported lead. Responses keep track of interactions with a contact or lead. The response will appear on the Responses tabs in the Detail views.

To add

  1. When importing leads, create an import action and click Define next to Add Response.
  2. Click the Lead Source Find button to find a lead source.
  3. Click the Response Date Calendar button to select the date.  
  4. Click the Status drop-down arrow and select an item from the list.
  5. Click the Response Method drop-down arrow and select an item from the list.
  6. If necessary, click the Campaign Find button to find a campaign. Your selection is added to the dialog box.
  7. Click the Stage drop-down arrow and select from the list.
  8. Click the Interest drop-down arrow to select a description for how the contact or lead responded.
  9. Click the Interest Level drop-down arrow to indicate how interested the contact or lead is.
  10. In the Comments box, type any notes.
  11. Click Save.
