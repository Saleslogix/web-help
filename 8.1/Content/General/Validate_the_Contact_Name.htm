Adding or Editing Name Information

Use the Edit Name dialog box to add or edit name information.This ensures that a name includes a first and last name and that the name is stored correctly in the database. The templates and mail merge features require the correct name, title and suffix.

To add or edit

  1. Open the Edit Name dialog box
    • From the Contact/Account Information or Contact Detail view, click the Contact Edit button.
    • From the Insert Lead, Lead Detail, or User Detail view, click the Name Edit button.
  2. Click the Prefix drop-down arrow and select an item from the list.
  3. In the First box, type the person's first name.
  4. In the Middle box, type the person's middle name.
  5. In the Last box, type the person's last name.
  6. If necessary, click the Suffix drop-down arrow and select an item from the list.

  7. Click OK.
