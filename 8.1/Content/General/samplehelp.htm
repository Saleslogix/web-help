Naming Convention Guidelines

When creating custom help for list views, you must use the following naming convention in order for your help to display properly:

 

entitylistview.htm - Example: accountlistview.htm

Where entity is the name of the entity associated with the custom list view

Tips for Writing a Help Topic

When creating a custom Web help topic for a List view, consider including an overview or introductory paragraph that briefly describes the types of records that will appear on the List view. You may also choose to include procedural steps for opening the List view, working with groups, or any other tasks that can be performed from the list view.
