Using the Sales Order List View

The Sales Orders List view displays all the sales orders that you have access to. You can use groups and lookups to narrow the list of records that display.

How Do I?

Open the Sales Order List view.
  • On the Navigation Bar, click Sales Orders.
Refresh list information
  • Click the Refresh button.

Use the Summary View

Work with records in a grid

Filter my list

Open the Sales Order Detail view.
  • If necessary, use the Lookup to locate a sales order, and then click on the sales order number.

View a group

Use common tasks

On the Task Pane, under Common Tasks, you can:

  • Save Records as a Group
  • Export to File
  • Promote to Dashboard
