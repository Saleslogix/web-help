Editing Product Information

Use the Edit Sales Order Item dialog box to edit a sales order's product information.

To edit

  1. Open the Sales Order Detail view, and click the Products tab.
  2. Click the Edit link for the product you want to change.

To edit the product in the grid, select the product you want to edit and select or enter information in the appropriate column.

  1. In the Edit dialog box, edit the information as appropriate.
  • In the Line box, type the product line information.
  • In the Description box, edit the product description.
  • In the Family box, select a product family.
  • Click the Price Level drop-down arrow and select the price level to be used.
  • In the Discount box, enter a discount percentage. The number is automatically changed to a percent. For example, "50" will be "50%". If you enter ".5", then it will be ".5%".

The Adjusted Price is automatically calculated.

  • In the Quantity box, enter the number of products.
  1. Click OK.
