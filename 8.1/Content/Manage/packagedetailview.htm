Using the Package Detail View

Use the Detail view to see, add, or edit information for a selected record. The Detail view consists of information boxes, tabs, and a user-defined middle section.

Your workspace may be different depending on your access rights or if your installation has been customized.

How Do I?

Open the Package Detail view
  • On the Navigation Bar, click Sales, Support, or Administration, and then click Packages to open the Packages List view. If necessary, use the Lookup to locate a package, and then click on the package name.

Use the Lookup to search for records

Switch to the List view
  • On the toolbar click the List View button, or click the current group tab.

Use the Task Pane group list

View a group

Use common tasks

On the Task Pane, under Common Tasks, you can:

  • Add to Group
  • Remove from Group

Information Boxes

  • Name - The product package name.
  • Description - Description for the product package.
  • Status - The status of the package. Click the drop-down arrow and select an item from the list.
  • Create Date - Displays the date the product package was created.
  • Created By - Displays the name of the user who created the product package.
  • Modify Date - Displays the date the product package information was last changed.
  • Modified By - Displays the name of the user who last changed the product package information.

detail view tabs

Each tab shows specific information you can use for package related activities. Click a link to see more information about each tab.

Products

user defined section

You can drag and drop tabs into the user defined middle section. The tab will remain there until it is moved back to the lower section.
