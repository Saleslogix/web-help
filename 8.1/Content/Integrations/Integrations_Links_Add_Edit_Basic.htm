Adding or Editing a Basic Integration Link

Use the Add or Edit Link dialog box to add or edit integration links. If an integration's authorization type is Basic, use this dialog box to add links.

How Do I?

Add a link
  1. On the Integrations Detail view, click the Links tab.

    The Links tab is available only for integrations with a synchronization method of Client-side Sync or Server Sync.

  2. Click the Add Link button.
    This button is only available if the integration link type is System or Admin.
  3. In the URL box, type the SData URL that Saleslogix uses to share information between systems.
    • Use one of the following formats:
      • If your implementation does not use the default port (port 80), use format http://server:port/sdatavirtualdir/application/contract/dataset/
      • If your implementation uses port 80, omit the port number from the URL.
        The Sync Service automatically strips the default port from the URL which causes matching issues and errors. Avoid this by using format http://server/sdatavirtualdir/application/contract/dataset/
    • Substitute values provided by your integration partner:
      •  Sdatavirtualdir - For example, sdata.

      •  application - your accounting application name. For example, accounts50.

      •  contract - the contract name. For example, GCRM.
      •  dataset - distinguishes a data set or operating company. For example, /-/
  4. In the Display Name box, type the name you want to display when users are linking records between Saleslogix and the integrated system.
  5. In the User Name box, type the name used to log on to the integration system.
  6. In the Password box, type the corresponding password used for the user to log on to the integration system.
  7. Click the Test Link button.
    The name of the button depends on the authorization type and link type.
  8. Click OK.
Edit a link
  1. On the Integrations Detail view, click the Links tab.

    		2.

    The Links tab is available only for integrations with a synchronization method of Client-side Sync or Server Sync.
  2. Click the Edit hyperlink for the link you want to edit.
    This link is available only if the integration link type is System or Admin.
  3. Make any changes.
  4. Click the Test Link button.
  5. Click OK.
