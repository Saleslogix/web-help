Log Items Tab

The Log Items tab displays the messages generated when a synchronization job is processed. Use the filters on this tab to define the logging information you want to view.

To view

  1. Open the Sync Job Detail view for the job that contains the items you want to view.
  2. In the Filter By Log Type list, select the type of error message you want to view.
  • Debug - displays detailed errors to help you troubleshoot problems.
  • Errors - displays information if a significant problem occurred, such as a loss of data or loss of functionality.
  • Informational - displays actions taken by the service.
  • Warnings - displays an event that occurred that is not necessarily significant, but may indicate a possible future problem.
  1. In the Filter By Resource Kind list, select the entity for the selected log type.
  2. Click the Refresh log items button.
  3. Click the Select link for a log in the grid. Detailed information displays in the Call Stack box.
