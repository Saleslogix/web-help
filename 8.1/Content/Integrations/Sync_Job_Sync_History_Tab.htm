Sync Job Sync History Tab

The Sync History tab on the Sync Job Detail view lists each synchronization history record that is associated with a sync job.

How Do I?

Open the Sync History tab
  • Open the Sync Job Detail view, and click Sync History.
Filter the sync history tab list
  1. Click the Filter By User Name Find button and select the user whose sync history records you want to display.
  2. In the Filter By Resource Kind list, select the entity for the selected log type.

This view contains the following columns:

  • Sync history Log Name -opens the Sync History Detail view.
  • User - the user name that made the change.
  • Resource - the type of resource that was synchronized.
  • Record - open the detail view for the record that was updated.
  • Sent From - the application that sent the change.
  • Received By - the application that received the change.
  • Severity - the type of information logged for the sync history item.
  • Log Create Date - when the sync history log item was created.
  • Link - the link used to synchronize data between Saleslogix and the external integration.
  • Integration - which integration was involved in synchronizing the information.
  • Status -The result of the synchronization. Possible statuses include:
    • Created - The record was created in the application that received the change.
    • Conflict-Won - The record was updated in the application that received the change as a result of conflict resolution rules.
    • Conflict-Lost - The record was updated in the application that received the change as a result of conflict resolution rules.
    • Deleted -The record was deleted in the application that received the change.
    • No change made - no change was made in the application that should have received the change due to errors.
