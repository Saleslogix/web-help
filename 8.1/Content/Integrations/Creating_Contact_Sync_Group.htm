Synchronizing Contacts with Google

In order to synchronize contacts, must:

  • Create and configure the contact sync group
  • Do one or more of the following:
    • Add Google contacts to Saleslogix.
    • Add Saleslogix contacts to Google.

Creating a Contact Sync Group

To synchronize contact information between Saleslogix and Outlook, you must designate a contact sync group. If the group does not already exist, then you must first create the contact sync group.

To create a contact sync group

  1. Open the Contacts List view.
  2. Select one or more records by pressing the CTRL or Shift key as you click each item.
  3. Right-click in the List view grid and select Save Records as Group.

    4. If you do not select any records, you will be asked if all records in the group should be used. If you click OK, all group records will be selected.
  4. Give the group a descriptive name. For example, SyncSaleslogix.

  5. Click OK.
