Related Documentation

In addition to this Help system, the following printed and online documentation is available:

  • Saleslogix Quick Reference card provides commonly used buttons, e-mail configuration options, and tips for working with groups and Outlook Integration. The Saleslogix Quick Reference card is accessible from the Help menu.
  • The Saleslogix Implementation Guide provides step-by-step instructions for installing Saleslogix. The Saleslogix Implementation Guide is available on the Saleslogix DVD.
  • The Administrator Help provides detailed instructions for managing user, teams, offices, Network bundles, and remotes. Some user management features are available in the Web Client, while full features and functionality are available in the Administrator.
  • The Saleslogix Mail Merge Help provides detailed information on the following: writing an e-mail message to a contact, writing an e-mail message, letter, or fax using a template, and using the Mail Merge engine to send correspondence to a group of contacts, accounts, etc. This help system also explains how to create, edit, and delete templates.

Help is available in each Saleslogix application. Additional resources and documentation are available to Partners and Customers with a valid technical support contract at the Support Portal Web sites:

  • Saleslogix Partner Support Portal Web site: http://www.saleslogix.com/partners.
  • Saleslogix Customer Support Portal Web site: http://www.saleslogix.com/customers.
