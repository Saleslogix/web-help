What's New In This Release

Saleslogix Web Client contains the following updates:

Version 8.1

  • Changes were made to reflect product rebranding.
  • Activities now allow multiple contacts or leads to be added using the Participants tab.
  • Ability to associate a contact with a Saleslogix user.
  • Outlook sync replaces Intellisync for synchronizing contacts and activities with Microsoft Outlook.
  • New Sync History List view allows users to review Integrations synchronization history, including Outlook Sync history.
  • Ability to compose an e-mail message in Outlook and select contacts from the Saleslogix Address Book.
  • New Dashboard widgets for Website, Web Feed, and SData.
  • Reporting now uses the Job Manager.
  • Export to Excel now uses the Job Manager.
  • Features for Administrators and those with appropriate role access:
    • Job Manager
    • Integrations
    • SData Sync management
    • Ability to associate a user with a contact
