Deleting an Event

You can delete any event you create from the Activity List view.

To delete an event

  1. Open the Activities List view.
  2. Click Events.
  3. If necessary, use the filters to narrow the list.
  4. Right-click the event you want to delete and click Delete Event.
  5. Click OK.
