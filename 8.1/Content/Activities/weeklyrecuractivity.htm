Recurring Activities, Weekly

You can choose to schedule an activity to occur once, daily, weekly, monthly or yearly.

To schedule an activity to recur weekly

  1. Schedule or edit an activity.
  2. Click the Recurring tab.
  3. Under This activity occurs, select Weekly.
  4. Select how you want the activity to recur.
    • If you want the activity to recur on a specific day of the week, select Every X Week(s), enter the number of weeks between recurrences, and then select the day(s) of the week that you want the activity to recur.
    • If you want the activity to recur only after the previous occurrences have been completed, select Every X Week(s) after completion, and then enter the number of weeks between recurrences.
  5. In the Start Recurring box, click the Calendar button, select a date, and then click OK.

  6. Do one of the following:

    • In the End After box, enter the number of times the activity should occur.
    • Click the End Recurring Calendar button, select a different date, and then click OK.
  7. When finished making all activity changes, click OK.
