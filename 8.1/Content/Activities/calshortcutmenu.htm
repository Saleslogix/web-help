Working with Calendar Shortcut Menus

Calendar shortcut menus enable you to schedule a new activity, or to work with scheduled activities. The shortcut menu options available vary depending on where the menu is accessed.

To schedule an activity

  • On any Calendar view, right-click anywhere except on an activity.
Menu Option Description
Schedule Phone Call Opens the Schedule Phone Call dialog box.
Schedule Meeting Opens the Schedule Meeting dialog box.
Schedule To-Do Opens the Schedule To-Do dialog box.
Schedule Personal Activity Opens the Schedule Personal Activity dialog box.
Schedule Event Opens the Schedule Event dialog box.

To work with a scheduled activity

  • On any Calendar view,right-click an activity.
Menu Option Description
Open Activity Opens the Edit Activity dialog box.
Complete Activity Opens the Complete Activity dialog box.
Delete Activity Deletes the activity.
Go To Allows you to open the detail view of any record associated with the activity.

To work with a completed activity

  • On any Calendar view,right-click a completed activity.
Menu Option Description
Edit History Opens the History Detail dialog box.

To work with an event

Menu Option Description
Open Event Opens the Edit Activity dialog box.
Delete Event Deletes the event.
