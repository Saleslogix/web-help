Working with the Activities View

Use the Activities view to manage your scheduled activities, events, literature requests, and confirmations. If you have additional access rights, you can also view other Saleslogix users' activities. Use the Summary view or Details pane to see additional information for each activity. You can use filters to narrow the list of records in the list. For example, if the time frame is next month, use the Time Frame filter to show only activities scheduled for next month.

Your security access determines what functions are available. Contact your administrator for any access rights changes. WebViewer users may not access full functionality.

The tabs that appear in the Activities view are:

My Activities All Open Past Due
Alarms Events Confirmations
Literature

How Do I?

Open the Activity List view
  • On the Navigation Bar, click Activities.

Work with records in a grid

Use the activity shortcut menu

View the summary pane
  1. Click Summary.
  2. To return to the List view, click List.

Use the Task Pane Filters to narrow the list

The available filters are based on the columns in the selected list view and the information contained in those columns. The number of records listed after a filter criteria only counts a recurring activity once, it does not include each occurrence separately.
Filters Example
  1. On the Navigation Bar, click Activities.
  2. Click the My Activities tab.
  3. Use the Task Pane Filters to narrow the list:
    • Expand the Activity Type filter and select Phone Call.
    • Expand the Time Frame filter and select Tomorrow.
      All of the calls you have scheduled for tomorrow are shown in the tab.
