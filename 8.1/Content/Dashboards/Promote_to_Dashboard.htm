Promoting a Group to the Dashboard

Promoting a group to the Dashboard sends all the group data to a group list widget on the dashboard.

To promote a group to a dashboard

  1. From a list view, click the group tab you want to send to the dashboard.
  2. In the Task Pane, click Promote to Dashboard.
  3. Click the name of the dashboard where you want to send the data, and click OK.
  4. Click OK.
  5. Click Welcome in the Navigation Bar and select the dashboard tab you selected in step 3.
    The data appears on the dashboard in a group list widget. The default configuration for the group list is 10 lines; however you can change this in the widget editor to display up to 50 lines.
