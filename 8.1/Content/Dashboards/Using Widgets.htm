Using Widgets

Saleslogix provides several out-of-the-box widgets. You can add, edit, and remove widgets from the dashboard, and you can rearrange widgets on the page with drag-and-drop.

Using pre-defined dimensions, which are a type of filter, you can display metrics for your data across ranges or from distinct values and display this data in one of the charting widgets. For example, you can configure a column chart widget to display all the opportunities closed and won, by owner and the total amount of the sale, or a pie chart depicting sales won by a team.

The list of available widgets is determined by your implementation. You may not have access to all widget types.

How Do I?

Add a widget
  1. Right-click the My Dashboard tab, and click Add Content.
  2. On the Add Content dialog box, click the Add button for the widget you want to add.
    For example, to add a pie chart widget, click Add under Pie Chart.
    The blank widget appears on the dashboard and the widget editor opens.
  3. Configure the widget settings, and click OK.
Edit an existing widget
  1. On the widget toolbar, click the Settings button.
  2. Make the changes necessary.
  3. Click OK.
    The widget displays the changes.

Configure the Group List widget
  1. On the Group List Settings dialog box, type a title for the widget.
  2. Select an Entity and a Group from the drop-down lists. For example, Opportunities, and Closed Won.
  3. Type the Number of Records you want in the list. For example, 10, 25, or 50.
  4. Type the number of records you want visible in the Visible Records box; 10, 25 or 50.
    For example, if you typed 50 records in step 3, but only want to see ten at a time in the list, type 10. The widget resizes to accommodate the number of visible records.

    You can use any number between 1 and the maximum number of records configured to display. The default maximum is 50.

  5. Click OK.
    The widget displays the list.

    To show or hide columns in a group list, right-click any column header and select the columns you want to add and clear the columns you want to remove.

Configure the Links List widget
  1. On the Link Settings dialog box, type a title for the widget.
  2. Click Add Link.
  3. Double-click the Icon URL box, and type the path to the icon you want to display for the link. For example, /google.jpg.

    		4.

    This URL must be a relative path to an image on the Web server.
  4. Double-click the Link URL box, and type the URL for the link you want to display. For example, http://localhost/SlxClient/Account.aspx.

    5. This path can be a relative path within the SLXClient portal, or an absolute link to an external site. The URL for an external site must include the http:// designator. For example, to access www.google.com the URL must read http://www.google.com. Otherwise the browser will search for http://<SLXClient site>/www.google.com.
  5. Double-click the Link Text box, and type the text you want to display as the link. For example, Google Search Page.
  6. Double-click the Target box, and type an identifier for the target. For example, if you want the link to open in a new browser window, type New Window. When you click the link, a new browser window/tab opens.

    		7.

    If you leave this field blank, the link opens in the active browser window/tab when clicked. If several links have the same target designation, each link replaces the other as you click them.
  7. Click Add Link to add another link, or click OK to close the editor.
    When you click OK, the widget refreshes to display the links.
Configure the SData widget
  1. In the Edit Widget Settings dialog box, type a title for the widget.
  2. Click the SData Feed drop-down arrow and select a feed from the list. Options include:
    • Saleslogix Dynamic Entity - allows access to standard entity resource types like contacts, accounts, and tickets.
    • Saleslogix System - allows access to non-standard resource types like activities and attachments.
    • Any accounting integration feeds that are available to you.
  3. Click Next.
  4. If you receive a message that the feed does not contain any resources, you must select a different feed. If a feed does not contain at least one resource, then there will not be any data to view.
  5. Click the Resource drop-down arrow and select the type of information you want to view from the list. For example, Account or Activity.

    The contents of the Resource list are based on the feed you selected.

  6. In the Where Clause box, enter the "where" statement that was provided to you. If a where clause was not provided, contact your administrator.
    If a where clause is not defined, all data that you have access to will be returned for the SData resource. This could result in a very large data set.

    Creating a where clause is recommended for advanced users only. For more information about creating a where clause, see Creating a Where Clause.

  7. Click Next.
  8. Select the columns from your SData Resource that you want to display in the widget.
    • Select one or more columns from the first list and click the > button. To add all columns, it is not necessary to select each column; just click the >> button.
    • Use the < and << to remove any columns that you do not want to see.
  9. In the Visible Records box, type a value between 1 and 100 to determine the maximum number of records to display on your dashboard.
  10. Click Finish.
    After the widget settings are saved, the widget displays records from the feed. Information from an SData feed is subject to security, so you can only see records you have permission to view. If the number of records returned is greater than Visible Records, then use the scroll bar.
Configure the Website widget
  1. In the Edit Widget Settings dialog box, type a title for the widget.
  2. In the Website Address box, type of paste the URL of the website you want to add to the dashboard.
    The URL should be in the format http://website.com

  3. Do one of the following:

    • Accept the Height as the default of 250 pixels.
      This is about the same size as a Group List widget with 10 visible records.
    • Change the Height to any number between 100 and 1000.
      For example, changing the height to 500 pixels would make the widget twice as tall, while changing the height to 125 pixels would make it half as tall.
  4. Click OK.
    The website is added to the dashboard. If the website appears to be blank, then either the Website Address is invalid or the website may be unavailable.
Configure the Web Feed widget
  1. In the Edit Widget Settings dialog box, type a title for the widget.
  2. In the Feed Address box, type or paste the URL of the feed you want to add to the dashboard.
  3.  Click the Show drop-down arrow and select one of the following:
    • Headline only - displays only the headline of each feed story.
    • Headline and summary - displays the headline and a summary of each feed story.
  4. In the Maximum of box, type a number between 1 and 25 to determine the maximum number of feed items to display on the dashboard.
    Depending on the web feed, fewer items may appear on the dashboard.The default number is 10 which is a common limit on many web feeds.

  5. Do one of the following:

    • Accept the Height as the default of 250 pixels.
      This is about the same size as a Group List widget with 10 visible records.
    • Change the Height to any number between 100 and 1000.
      For example, changing the height to 500 pixels would make the widget twice as tall, while changing the height to 125 pixels would make it half as tall.
  6. Click OK.
    The web feed is added to the dashboard.
Minimize a widget
  • On the widget toolbar, click the Minimize  button.
    The widget content is minimized in the dashboard. Click the button again to maximize the widget.
Remove a widget
  • On the widget toolbar, click the Close  button.
    The widget is permanently removed from the dashboard.
