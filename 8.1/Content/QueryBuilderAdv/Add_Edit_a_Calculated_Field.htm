Adding and Editing Calculated Fields

 System Administrator Only

To open the Manage Calculated Fields dialog box

  1. Open the Query Builder.
  2. Click the Calculations button.
To add
  1. Open the Query Builder, and then click the Calculations button.

This button only appears if you are logged on as Admin.

  1. Click Add.
  2. Click the Properties tab.
  3. In the first Name/Alias box type a name that describes the purpose of the calculation.

The calculation name appears only in the Calculation Manager.

  1. In the second box type a name to be used in the calculation. It cannot contain any spaces or symbols. This is a required field. You will be unable to create a calculated field until this has been set.

When the calculated field is used in constructing a plugin or query, this name is displayed in the list of available fields.

  1. Click the Base Table drop-down arrow and select the main table used to create the calculated field.

If you are joining two tables, this is the parent table.

  1. Click the Calc. Type drop-down arrow and select a Calculation Type.
  2. Type a Description if you want to have a longer description of the field available.
  3. Click the Calculation tab.
  4. Determine if the values you want to use for the calculated field are in the selected Base Table. 

The table displayed in the upper list view is the table you selected in the Properties tab.

  1. If the base table does not contain the values you need, change the base table rather than creating a join. Click the Properties tab and select another base table.
  2. Create the calculation.
  3. Click + or - to expand or collapse the list of tables and fields to find the first field for the calculation.
  4. To add a field, click the field in the upper box to add it into any position within the calculation.

This is the first field in the calculation.

  1. Do one of the following:
  • Click an operator if you are creating a numeric calculation.

You must insert an operator between fields in a numeric calculation.

  • For a string, you do not need to pick an operator, but you may want to insert a space for the sake of appearance.
  1. Click another field in the tree view to add it to the lower tab view or enter an integer.
  2. Continue adding fields as necessary to build the calculated field.
  3. Use brackets to group calculations. To insert a bracket, place your cursor in the calculation where you want to insert a bracket and click a bracket button. You must have both a beginning and ending bracket.
  4. Click OK.
  5. Click Close.
To edit
  1. In the Manage Calculated Fields dialog box, select the calculation, and then click Edit.
  2. In the Edit Calculation dialog box, click the Calculation tab.
  • To add a field, click the field in the upper box to add it into any position within the calculation.
  • To add an operator or a bracket, position the cursor within the calculation and click an operator or bracket button. An operator is necessary between fields in a numeric calculation and brackets must have a beginning and an ending bracket.
  • To delete any item within a calculation, highlight the item and press Delete.
  1. Click OK.
  2. Click Close.
