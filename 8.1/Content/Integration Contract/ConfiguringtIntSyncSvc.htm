Configuring the Integration Synchronization Service

The Integration Synchronization Service (Saleslogix SData Sync Service) allows data transfer between Saleslogix and your integrations. The service must be started and set to run as automatic.

Synchronization of information in real-time requires an internet connection. When using Accounting Integration on the Offline Web Client, data in real-time data views will be current as of the last time the Client connected with the host database.

To configure

  1. Log on to the Administrative Workstation as the WebDLL user.
  2. Make sure you configure and enable SData synchronization and your integration(s) before starting the service.
  3. Ensure you have enabled Windows Authentication for the WebDLL user.
  4. On the Administrative Workstation, open Services.
  5. Right-click Saleslogix SData Sync Service, and then click Properties.
    1. In the Startup type box, select Automatic.
    2. Set the service to log on and run as the WebDLL user.
    3. Click Start.
  6. Click OK.
