Creating a New Account from a Sales Order

You can create a sales order before you create an account or contact. If the prospect decides to do business with your company, you can create a new account and contact from the sales order with limited company and contact information.

To create

  1. Create a new sales order.
  2. Click the Add new account button.
  3. In the Quick Insert Account Contact dialog box, add the following:
    1. In the Account box, type an account name.
    2. In the Type box, select the account type.
    3. In the Main box, type the main office phone number.
    4. In the Address box, add the account's business address.
    5. In the Contact box, type a contact name.
    6. In the Title box, type or select the contact's title.
    7. In the Main box, type the contact's phone number.
    8. In the Mobile box, type the contact's mobile phone number.
    9. In the E-Mail box, type the contact's e-mail address.
    10. Click OK.
  4. Click Save.
