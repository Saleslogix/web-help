Select an Accounting System

The Select an Accounting System screen in the Link to Accounting wizard displays the accounting systems available to link with the selected account.

To select

  1. Open the Link to Accounting wizard.
  1. Complete the Select an Accounting System screen.
    BoxDescription/Action
    SystemSelect the accounting system with which you want to link the account.
  2. Click Next to go to the next step.
