Security Tab

This functionality is only available when the appropriate license is installed. For complete information about licensing, see the Saleslogix Planning Guide.

Use the Security tab on the User Detail view to define a user's default field level security profile. The system administrator always has access to everything, and the system administrator’s security access cannot be changed.

To set options

  1. In the User Detail view, click the Security tab.
  2. In the Field Level Security Profile box, select the default profile to use when this user is added to a team or another user's team. Security profiles are created in the Administrator.
    3. Tell me more...

    When User A is manually added to User B's team, (using the User Team Membership tab), User A can access all of User B's records.  However, User A's Field Security permissions will determine if User A has the ability to modify User B's records.  Users always have read/write access to all individually owned accounts, or to accounts owned by Everyone.

  3. Click Save.
