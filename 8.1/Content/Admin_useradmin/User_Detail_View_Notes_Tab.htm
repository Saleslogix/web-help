Notes Tab

Use the Notes tab on the User Detail view to add notes about the user. This tab allows you to add information that may not be captured on another tab.

To manage

  1. In the User Detail view, click the Notes tab.
  2. To add a date/time stamp, click .
  3. Type notes about the user.
  4. Click Save.
