Using the Defects Tab

The Defects tab on the Ticket and Return Detail views displays all associated defects. Use this tab to add or remove an association with a defect.

The tab displays the read-only information of Defect ID, Reported Date, Priority, Severity, Status, Resolution, and Description.

How Do I?

Open the Defects tab
  1. Open the Ticket or Return Detail view.
  2. Click the Defects tab. If the tab is not visible, click the More Tabs tab.
View defect details
  • Do one of the following:
    • To open the Defect Detail view, click the defect number.
    • To view the defect description, select the record in the grid. The defect Description appears in the box in the tab view.
Associate a defect
  1. Click theAssociate Defect button.
  2. Locate and select the item using the lookup.
  3. Click Add Selected.
Disassociate a defect
  1. Select the record you want to disassociate and click the Remove button.
  2. Click OK.
Add a new defect (Ticket Detail view only)
  1. Click the Create Defect button.
  2. Follow the steps for adding a defect.
