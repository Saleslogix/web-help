Adding/Editing an Alternate Address

You can use the Address dialog box to change address information. You can have as many custom addresses as necessary for a contact or an account. However, you may select only one primary billing address and only one primary shipping address.

To open the Address dialog box

  • Do one of the following:
  • On the Account, Contact or Lead Detail view, click Addresses.If the tab is not visible, click the More Tabs tab, and then select the tab from the list. Click the Edit link of the address you want to modify, or click the Add Address button to add a new address.
  • Click the Address Edit button on the Contact/Account Information or Insert Lead view.
  • Click the Address Edit button on the Contact, Account or Lead Detail view.
  • Click the Address Edit button on the Sales Order Detail view, on the Bill/Ship Address tab.

To add or edit

  1. In the Description box, click the drop-down arrow, select a description for the address, and then click OK.
  2. Select or clear the Primary,Primary Billing, or Primary Shipping check boxes as necessary.
  3. Enter or edit the address information in the address boxes.
  4. In the City, State, or Country boxes, click the drop-down arrow and select an item from the list, and then click OK.
  5. Click in the Postal Code box and enter any changes.
  6. Click in the County box and enter any changes.
  7. Click in the Attention box and enter the salutation to use for mailed correspondence.
  8. Click OK.
