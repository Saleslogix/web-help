Editing Mail Merge Output Prior to Sending

You can edit the merged document for individual recipients prior to delivery. For example, you can add a personal greeting to a single letter. This option is found on the Merge Options tab for Mail Merge.

To edit mail merge output prior to sending

  1. On the Write menu, select Mail Merge.
  2. On the Merge Options tab, complete the Template, Merge With, and Output To boxes.
  3. In the Merge Options area, select the first option named Edit individual <document> prior to sending.
  4. Complete other information boxes as needed, and then click Merge.

The merge engine will show progress, and you may have to select a contact address.

  1. In the Edit Merged Documents dialog box, select the document to edit, and then click Open.
  2. In the editor, make changes or add a greeting to the document.
  3. On the toolbar, click Save and Close.
  4. (Optional) Select another document, click Open and follow steps 6 and 7 for each document.
  5. When all document edits are complete, click Done.

The merged documents process according to your output selection.
