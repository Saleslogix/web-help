Library Document Properties

Use the Document Properties dialog box to add to or change the properties of files in the Library. In addition to the editable fields, this dialog box provides the following read-only information:

  • Directory: The location where the document is saved.
  • Status: Applies to Remote users only. The status can be Available, Ordered, Delivered, Not Read, Read, or Revised.
  • Created: The date the file was added to the library.
  • Size: The size of the file.

The system administrator and users with the appropriate role can access these features.

To view or set Library document properties

  1. On the Navigation Bar, click Library.
  2. In the list, select the Library document you want to edit, and click the File Properties button.
  3. Use the Revised box to change the date and time the file was last edited so that synchronization sends this file to the Remote Clients.
    This field does not update automatically.
  4. Clear the Do Not Expire check box to activate the Expires box.
    This check box is selected by default.
  5. Set a validation date in the Expires text box. This option can be used for time sensitive documents such as price sheets.
  6. Use the Description text box to include up to a 255-character comment about the file.
    The file description appears in the Library details view.
  7. Use the Abstract field to include up to a 1024-character comment about the file.
    The abstract can only be viewed in this dialog box.
  8. Check the Force distribution of this file option to automatically send files and file revisions to all Remote users.
  9. Click OK.
