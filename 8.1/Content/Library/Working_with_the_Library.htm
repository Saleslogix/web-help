Working with the Library

The Library is a central repository for company information (in the form of documents, URLs, and so on) that provides an easy way to distribute information to remote users. Files are organized into folders and sorted by name. Click the column headings to sort the library files list.

Using the Saleslogix Web Client, you can request and download a Library file when you need it. The Library displays the file name, size, creation, revision and expiration dates, and a brief description of the file.

Your company’s library may include product information, policies and procedures, presentations, Web addresses, and so forth.

Only the administrator and an administrative user can add files to the Library.

How Do I?

Open the Library
  • On the Navigation Bar, click Library.
Download a Library file
  1. In the tree view, click the "+" signs to expand the folder that contains the file you want to open.
  2. In the list, click the File name of the file you want to download.

  3. Do one of the following:

    • Open the file
    • Save the file

If your browser opens the file instead of saving it to a location on your computer, change your default browser settings to automatically save files instead of opening them. Alternatively, hold down [Shift] and then click the link to download without opening all files except for those with the extension HTML. HTML files are opened in another window.

The system administrator and users with the appropriate role can access these features.The system administrator and users with an administrator role can access these features.
Add a document to the Library
  1. Select the folder under which you want to add the item.
  2. Click Add Files.
  3. Select the file(s) you want to add.
  4. Click Open.
Add a document to the Library using drag and drop

  • You may not have access rights to this feature. For more information, please contact your administrator.
  • If you are using Internet Explorer, this feature requires the installation of the Desktop Integration Module.
  1. Browse to the file you want to add.
  2. On the Navigation Bar, click Library.
  3. In the tree view, click the "+" signs to expand the folder to the location where you want to add the file.
  4. Drag the file and drop it on the Library view.
  5. In the Description box, enter a description for the file if the default description is not clear.
  6. Click OK.

You cannot drag and drop a folder and its contents to the Library. You can only drag and drop individual files.
