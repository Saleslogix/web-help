Campaign Target and Response Filters

Use the Show Filters options to limit the target contacts and leads that display in the list.

To filter

  1. Click Show Filters to expand the list of filters.
  2. Select the check box(es) of the filter(s) you want to apply.
    • Contacts - Displays contact responses that meet the filter requirements.
    • Leads - Displays lead responses that meet the filter requirements.
    • Responded - Select Yes to view targets who have responded or No to view targets who have not responded.
    • Status - Click the drop-down arrow and select a status from the list. The list will display only the targets with the selected status.
    • Group - Click the drop-down arrow and select a group from the list. The list will display only the targets in the selected group.
    • Priority - Click the drop-down arrow and select a priority from the list. The list will display only the targets with the selected priority.
    • Last Stage - Click the drop-down arrow and select a stage from the list. The list will display only the targets within the selected stage.

    You may select more than one filter option. For example, you can check the Show Contacts and Response Method options to view contact responses that have the same response method.
  3. If necessary, click the Refresh button to update the grid.
