Adding Lead Sources to a Campaign

You can add lead sources to a campaign using the Add Campaign Lead Source dialog box. Use the filter to narrow the list of lead sources in the Available Lead Sources tree view. You add lead sources by selecting them from the Available Lead Sources tree view and adding them to the Selected Lead Source grid. If you add a lead source in error, you can remove it.

To add

  1. From the Campaign Detail view, click the Lead Sources Find button.
  2. Use the Lookup By to narrow the list of lead sources . Enter the filter criteria using the drop-down list and text box, and then click Find.
  3. Click Associate next the lead source you want to associate with the campaign.
  4. Repeat steps 1-3 until you have added all the lead sources to be associated with the campaign.
