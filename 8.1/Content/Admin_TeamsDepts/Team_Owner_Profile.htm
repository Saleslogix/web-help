Team Owner Profile

The Team Owner Profile is the only security profile that can change ownership. Any number of team members can be team owners.

It is also possible for a team to have no owners. If you change the profile of users that have the "Team Owner Profile" assigned to another profile, then only the system administrator can change the ownership of accounts.
