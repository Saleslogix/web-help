Locations Tab

The system administrator and users with the appropriate role can access these features.The system administrator and users with an administrator role can access these features.

The Locations tab in the Secured Action Detail view displays where the secured action is used. The file name and server path from the Web site displays for each element. This information is read-only.

When you delete or rename a secured action, the secured action is de-coupled from the element in the Application Architect. Before modifying the secured action, Saleslogix recommends you save the information found on this tab as a history record. You can copy and paste the information to a text file.

If necessary, you can use the information on this tab to recreate a secured action or to remove the action from elements in the Application Architect.
  • The first time you access this tab, information may take a long time to load.
  • If you make changes to secured actions, the information on this tab will refresh when you rebuild and deploy your Web site.
