Editing a Template

You can edit private or public templates; you can edit a public template, if you copy the template and save it as a private template.

If you are editing the template during Mail Merge, when you are finished with your edits, you must Save and Close the editor to allow the merge to continue.

To edit a template

  1. On the Write menu, select Templates.
  2. In the Manage Templates dialog box, select a template from the list of Private or Public Templates, and then click Edit.
  3. If you are editing a public template, when the Warning message appears, click Yes to copy the template.
  4. In the properties dialog box, do one of the following:
    • Set the properties for the template, and then click OK.
    • Leave the properties as indicated, and then click Close.
      The editor opens with the template's standard text and merge fields visible.
  5. To make changes to the template, you can type additional text in the template or use the Saleslogix toolbar to:
    • Insert Saleslogix merge fields to the template
    • Preview the template
    • Attach files to the template if it is an E-mail type template
  6. When all edits are complete, on the Saleslogix toolbar, click Save and Close.
