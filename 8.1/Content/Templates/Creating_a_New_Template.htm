Creating a Template

Using the template editor, you can create a private template.

To create a new template

  1. On the Write menu, select Manage Templates.
  2. In the Manage Templates dialog box, click New.
  3. In the properties dialog box, set the template properties, and then click OK.
    A sample, base template appears in the editor. For example, if you are creating a letter template, the sample, base letter template appears in the editor.
    You can type or remove content, or paste text from another document.
  4. To add Saleslogix merge fields to the template, place the cursor in the template where you want the merge field to appear, click Insert SLX Field, then from the merge field list, select a field.
    The new merge field appears in the template.
  5. Continue adding or deleting merge fields as needed.
  6. To save the template without closing the editor, click Save.
  7. In the editor, use the Saleslogix toolbar to preview the merged template, to attach files, or to perform other functions.
  8. When finished, click Save and Close to save the template and close the editor.
