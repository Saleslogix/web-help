Releasing a Private Template

If releasing a Private Template to another user, it is recommended that you copy the template first, then release the copied template to the other user. That way, you will still have the Private template in your list.
  1. On the Write menu, select Templates.
  2. In the Private Templates list, click to expand the Letter, Fax, or E-mail folder.
  3. Right-click the template that you want to release to another user, and from the shortcut menu, point to Properties.
  4. In the Properties for dialog box, in the Owner box, click the Lookup to select a Saleslogix user from the list. Double-click the user's name.
  5. Click OK. The template will now appear in the Private Templates folder for the user selected in Step 3 and will be removed from the current user's Private Templates list.
