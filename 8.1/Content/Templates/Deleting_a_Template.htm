Deleting a Template

You can only delete private templates that you own. However, if the template has been released to others and is shared, your Saleslogix administrator should remove the template.

To delete a template

  1. On the Write menu, select Templates.
  2. Open the Manage Templates or Select a Template dialog box.
    • On the Write menu, select Templates.
  3. In the Manage Templates dialog box, select a template from the list of Private Templates, and then click Delete.
  4. Click OK to close the message box.
  5. In the Manage Templates dialog box, do one of the following:
    • Select another private template to delete.
    • Click Close to close the dialog box.
