Query Builder Layout

Use the Query Builder Layout tab to specify which fields are displayed and how the data appears. You can add, edit, delete, or reorder the fields in the layout. You can also set the column name, width, and format of each field. The Layout tab must include at least one field or no data will appear.

The Query Builder supports many features. The tabs you see depend on the feature you are using.
To add a field in a layout
  1. Open Query Builder.
  2. Click the Layout tab.

    The default layout for new groups is determined by the group selected as the default group.

  3. Select a table.
  4. Select a field.

You can scroll the layout list to show the field as it is added to the layout.

  1. Select the field and click Edit.
  2. Complete the Assign Query Layout dialog box.
To edit a field in a layout
  1. Open Query Builder.
  2. Click the Layout tab.
  3. Select a field from the grid.
  4. Click Edit.
  5. Make changes in the Assign Query Layout dialog box.
To delete a field in a layout
  1. Open Query Builder.
  2. Click the Layout tab.
  3. Select a field in the grid.
  4. Click Delete.
To reorder fields in a layout
  1. Open Query Builder.
  2. Click the Layout tab.
  3. Select a field. Do one of the following:
  • Drag and drop the field to the new position.
  • Click Move Left or Move Right until the field is in the correct position.
