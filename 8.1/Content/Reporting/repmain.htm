Report Preview

You can only see reports that have been released to you or to a team that you belong to. You cannot save reports in the Saleslogix Web Client.

You must allow pop-ups before you can export or print a report.

How Do I?

View or print a detail report from a detail view

View or print at report from the Report History tab
  1. On the Navigation bar, click Reports, and then click the History tab.
  2. Locate the report you want. You can sort by grid column headings.
  3. Click the report.
  4. The report opens in the viewer native to the file type. For example, Acrobat Reader for PDF formats.
