Return Information Boxes

The following table describes the default information boxes.

Box Description/Action
Return ID Automatically displays the next available Return ID. You cannot change the ID. Not available when adding a return.
Ticket ID Displays the ID of the ticket from which this return was generated. Click the Find button and edit using the lookup.
Account Displays the account associated with the ticket.
Contact Displays the name of the Contact from which this Return was generated. Click the Find button and edit using the lookup.
Reason Type the reason why the item is being returned.
Customer PO Type the purchase order number for the return.
Status Displays the status of the return.  For example, Open, Closed, Pending, or In Process.
Type Return type assigned to the return.
Priority Priority assigned to the return.
Expected by Date by which  the product is expected to be returned. To edit, click the Calendar button to select the date.  
Assigned To The user assigned to the return. Click the Find button and edit using the lookup.
Created By

Displays the user who closed the return. Click the Find button and edit using the lookup.
Closed By

Displays the user who closed the return. Click the Find button and edit using the lookup.

The date and time it was closed. To edit, click the Calendar button to select the date.
