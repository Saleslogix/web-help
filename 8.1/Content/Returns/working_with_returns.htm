Working with Returns

Returns are agreements to accept returned products from a customer for service, replacement, refund, or credit You can use the Return Detail view to work with a single return, or the Returns List view to work with returns in a spreadsheet-type format.

Here are some of the features that are available to help you manage your returns:

  • Use the Return Detail view to add or edit information within a return.
  • Use the Return List view to view and manage return groups.
