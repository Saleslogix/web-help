Definitions

The Definitions tab contains a list of pre-defined job types. Select a job type from those listed and use the schedule wizard to set a time and recurrence for running the job.

Note: New job types are created in the Saleslogix Application Architect.

To create a schedule for a job

1. Select the job, right-click and then click Create Schedule.

2. In the Run/Schedule Job dialog box, make your modifications.

3. Click OK.
